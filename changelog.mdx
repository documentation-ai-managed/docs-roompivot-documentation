## Latest Release

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added batch processing: Upload multiple room photos at once and generate staged versions in parallel.
- Introduced custom style packs: Select from predefined themes like `modern-minimal`, `boho-chic`, or `industrial-loft`.

## Improvements

- Reduced generation time by 40% with optimized AI models.
- Enhanced image quality for better lighting simulation and furniture placement accuracy.

## Bug Fixes

- Fixed rendering artifacts on high-resolution uploads `>4K`.
- Resolved export failures when using `{API_KEY}` authentication.

</Update>

## Upgrade Guide

Follow these steps to update RoomPivot and access the latest features.

<Steps>
  <Step title="Check Current Version" icon="check-circle">

    Use the dashboard or API to verify your version.

    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const response = await fetch('https://api.roompivot.com/v1/version', {
        headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` }
      });
      const data = await response.json();
      console.log(data.version); // e.g., "1.2.0"
      ```
      ```bash
      curl -H "Authorization: Bearer YOUR_API_KEY" \
           https://api.roompivot.com/v1/version
      ```
    </CodeGroup>

  </Step>
  <Step title="Update SDK" icon="download">
    Install the latest SDK package.

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm update @roompivot/sdk@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn upgrade @roompivot/sdk@latest
        ```
      </Tab>
    </Tabs>

  </Step>
  <Step title="Test New Features" icon="play-circle">
    Run a test redesign on a sample room photo to confirm everything works.
  </Step>
</Steps>

## Known Issues and Resolutions

<ExpandableGroup>
  <Expandable title="Slow generation on mobile networks" default-open="true">
    Network throttling affects AI processing. Use Wi-Fi or upgrade to premium tier for priority queuing. Resolution planned in v1.2.1.
  </Expandable>
  <Expandable title="Custom furniture library import errors">
    Large ZIP files `>50MB` fail. Compress assets and retry. Fixed in batch processing feature (v1.2.0).
  </Expandable>
</ExpandableGroup>

<Callout kind="tip">
  Report issues via the feedback form in your dashboard. Include room photo examples and browser details for faster resolutions.
</Callout>
